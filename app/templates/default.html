Como criar Skills

Skills expandem a capacidade do agente ao responder consultas. Cada skill ativa é injetada no contexto do Deep Agent como conhecimento especializado.

Exemplos de uso:

Análise Financeira — regras para interpretar DRE, balanço patrimonial, indicadores de liquidez, rentabilidade. O agente passa a calcular e explicar índices financeiros automaticamente.

Segmentação RFM — instruções para calcular Recência, Frequência e Valor Monetário a partir de dados transacionais, classificar clientes e gerar recomendações.

Análise de Churn — métricas de retenção, cohort analysis, indicadores preditivos, formato de relatório com ações recomendadas.

KPI Dashboard — definição de KPIs específicos do negócio, fórmulas de cálculo, metas, semáforos de performance.

Use Markdown no conteúdo: cabeçalhos, listas, exemplos, fórmulas. Quanto mais contexto e exemplos, melhor a interpretação do agente.
